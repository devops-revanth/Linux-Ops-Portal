# Linux Operations Portal

Enterprise web application for Linux Infrastructure Operations.

LOP replaces Excel-based inventories with a centralized portal for Linux server inventory, patch management and automation.

---

## Features

- Dashboard
- Inventory Management
- Server Details
- Settings
- Patching
- Global Search
- PostgreSQL Backend
- Docker Deployment

---

## Technology Stack

- Flask

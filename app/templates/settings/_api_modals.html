{# ── Regenerate Token confirmation ─────────────────────────────────────── #}
Regenerate Token

The current token will be immediately invalidated. Any Ansible playbooks using the old token will fail until updated.

{# ── Revoke Token confirmation ──────────────────────────────────────────── #}
Revoke Token

Revoke the active API token? Ansible pushes will be rejected until a new token is generated.
